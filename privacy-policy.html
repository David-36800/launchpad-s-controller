Privacy Policy — Launchpad S Controller

Last updated: March 2026

🇬🇧 English

Data Collection

Launchpad S Controller does not collect, store, transmit, or share any personal data whatsoever.

How it works

This extension communicates exclusively with a local WebSocket server running on your own machine (localhost:8765). No data leaves your computer. No external servers are contacted. No analytics, no tracking, no cookies.

Permissions

The extension requires the following permissions:

  • tabs — to detect and interact with open YouTube tabs
  • scripting — to simulate button clicks on the YouTube interface
  • host_permissions (youtube.com) — to execute scripts on YouTube pages only

These permissions are used solely to control YouTube playback locally. No data from these pages is read, stored, or transmitted.

Contact

If you have any questions, please open an issue on the project repository.

🇫🇷 Français

Collecte de données

Launchpad S Controller ne collecte, ne stocke, ne transmet et ne partage aucune donnée personnelle.

Fonctionnement

Cette extension communique exclusivement avec un serveur WebSocket local tournant sur votre propre machine (localhost:8765). Aucune donnée ne quitte votre ordinateur. Aucun serveur externe n'est contacté. Pas d'analytique, pas de tracking, pas de cookies.

Permissions

L'extension requiert les permissions suivantes :

  • tabs — pour détecter et interagir avec les onglets YouTube ouverts
  • scripting — pour simuler des clics sur l'interface YouTube
  • host_permissions (youtube.com) — pour exécuter des scripts sur les pages YouTube uniquement

Ces permissions sont utilisées uniquement pour contrôler la lecture YouTube en local. Aucune donnée de ces pages n'est lue, stockée ou transmise.

Contact

Pour toute question, ouvrez une issue sur le dépôt du projet.
